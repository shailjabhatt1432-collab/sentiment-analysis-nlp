# Sentiment Analysis using NLP

This project performs sentiment analysis using NLP techniques.
It classifies text into Positive, Negative, and Neutral sentiments.

## Tech Stack
- Python
- NLP
- TF-IDF
- Logistic Regression
- Streamlit

## Accuracy
~96%
